# Week 4 Calculator App

### Info : Creating Calculator App with SwiftUI 

### Function: 
(1) 사칙연산(덧셈, 뺄셈, 곱셈, "="으로 결과값 출력) 
(2) 입력수정: 사용자 입력을 하나씩 삭제 가능한 "<" backspace 추가 
(3) 초기화: 입력 한번에 초기화 가능한 "C" button 추가
